迁移后的开发注意事项

  1. ToastUtil
  2. LazyOnClickListener
  3. IOSStyleToggle
  4. 格式化字符串的处理（String.format）
  5. DialogManager
  6. 字符串工具类（utils.string）下替代JudgmentLegal
  7. xml的tools命名空间
  8. live templete快捷键
  9. ApplicationContext
  10. FilePathManager
  11. ImageUtil
  12. HttpUtil
  13. MobileApi
  14. simpleXml的注解和@ReqParam
  15. BaseActivity：
    1. navigate系列
    2. toolbar系列
    3. setPageXXX系列
    4. initXXX系列
    5. regPresenter
  16. MVP框架
    1. IBaseCoreView/IBaseUIView/IListView
    2. presenter 调用Model；存储控制标记，页面数据；onDestory;onSuccess/onFailure
    3. model 构造函数注册接口
  17. 图片上传套件UploadFileView
  18. CommonLibrary是汇旺和三维度项目共用，需要专门维护，只要改了CommonLibrary及以下的module里的代码就必须:
    1. 写rb/rl（live templete快捷键）注释
    2. 文件头写修改记录
    3. 告诉大家
    4. 建议提交前先知会本人
  19. DevTool
  20. 尽最大可能不要再修改Constant/URL/GlobalVariable
  21. AppInfoUtil
  22. UserManager
  23. MoneyFormatter/StringFormatter/StringValidator/StringConverter
  24. PullableLayout和RecyclerView
  25. RecyclerBanner
  26. BaseRecyclerAdapter
  27. SimpleImageAdapter
  28. 禁止使用废弃类和废弃属性
  29. 文件头和修改注释
  30. BaseFragment 和 v4 Fragment
  31. StaticFragmentPager 静态的Fragment的ViewPager
  32. BasePullableActivity 和 BasePullableFragment
  33. layout 里 控件id 必须按照格式书写 e.g. btn_sign_in_submit;对应java代码里控件的变量名必须与控件id一致
